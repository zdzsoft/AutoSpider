AutoSpider（网络爬虫工具）

简介：
   AutoSipder是一个纯Java爬虫程序，用户可以使用它来从网上抓取想要的资源。使用 Java 编写脚本，提供强大的自定义抓取采集内容，采集规则进行了精心设计，规则灵活简单、高度封装，用于通用方法集与大量Demo，让你轻松添加规则；
   (1)支持数据自动导出，导出格式为：文本、Excel等；
   (2)支持命令行操作界面，适用于各种运行环境；


参考样例：
    com.zdzsoft.spider.GongPingJia.java， 自动抓取二手车网站公平价的数据，并导出成Excel文件car.xls。
